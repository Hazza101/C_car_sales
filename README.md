# C Car sales system
This was a first year university project to learn the basics of programming in C. It is a terminal based application for customers to buy cars and for employees to look at sales data.

## Features
- Main menu to choose what you wanna do
- View Cars for sale
- View sales data
- Buy a car
- Exit cleanly
